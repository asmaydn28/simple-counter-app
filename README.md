# Simple Counter App 🕐

A simple JavaScript counter application. You can increase the counter by clicking a button.

## 🚀 Features
- Initial value: `0`
- Increases by `+1` on each click
- Uses clean and simple HTML, CSS, Bootstrap, and JavaScript
